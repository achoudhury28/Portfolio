IT-260 — Advanced Programming (C#) Deep dive into object-oriented programming, data structures, algorithm design, and performance optimization.
⌄

Course Description

The purpose of this course is to study the fundamental concepts and techniques necessary to write high-quality programs, including basic and advanced concepts of object-oriented approach, modular design, inheritance, GUI objects, exception handling, files and streams, and class libraries. All of the mentioned concepts and techniques are studied using the C# programming language. This course explores important interactions between C# and the .NET Framework using Microsoft Visual Studio as an Integrated Development Environment (IDE).

Course Reflection

IT-260 strengthened my ability to write high-quality programs by focusing on object-oriented development, modular design, and building applications with real-world structure. The course challenged me to go beyond simple console programs and think like a developer who designs software for usability, maintainability, and long-term improvement. Learning C# in Microsoft Visual Studio helped me understand how professional developers build, debug, and organize code using the .NET framework.

A major takeaway from this course was learning how to break a large problem into smaller, manageable components using classes, methods, and reusable modules. This helped me build programs that are easier to test and update. I gained deeper experience with inheritance, polymorphism, and exception handling, which improved how I manage unexpected errors and edge cases. These topics taught me that good code is not just code that works, but code that works reliably and can be understood by others.

This course also improved my critical thinking and problem-solving skills. I practiced analyzing program requirements, selecting the correct approach, and designing logic flows before writing code. Debugging became a structured process rather than guessing. I learned to identify root causes, test multiple solutions, and validate outputs using controlled inputs. These skills translate directly into backend development and data engineering, where accuracy and reliability matter.

Overall, IT-260 helped me build confidence in writing professional programs and strengthened my mindset around clean architecture and organized design. It showed me how to use programming principles to solve real problems and reinforced the importance of documentation, naming conventions, and readable code. The technical foundation I gained supports my long-term focus in backend development and data engineering by improving how I design scalable and maintainable systems.

⌄

Course Description

This course introduces the fundamental concepts of database management systems, with emphasis on the relational model, which is an increasingly important and widespread area in the computer field. Demonstrations and hands-on practice using ORACLE: SQL and PL/SQL reinforce the fundamental concepts.

Course Reflection

IT-330 introduced me to the principles of designing and managing databases, focusing on the relational model and how organizations store and retrieve data efficiently. The course emphasized database structure, data integrity, and using SQL to solve real business questions. Learning Oracle SQL and PL/SQL helped me understand how databases support applications and decision-making in real environments.

One of the biggest lessons from the course was learning how to design databases correctly using ERD modeling and normalization. I learned to reduce redundancy and prevent inconsistencies by applying normalization concepts. I also gained strong SQL skills such as joins, aggregations, subqueries, and creating tables with constraints. These skills are essential for backend development and data engineering where data accuracy and performance matter.

This course strengthened my analytical thinking by requiring me to translate business requirements into database solutions. I practiced debugging SQL queries, improving query performance, and ensuring data integrity using primary keys, foreign keys, and constraints. I learned how design decisions impact reliability and performance when working with larger datasets.

Overall, IT-330 helped me build confidence working with relational databases and improved my ability to design efficient schemas that support applications and reporting. The course directly supports my career goal in data engineering because data engineers must build reliable data structures, validate data quality, and optimize query performance for analytics and backend systems.
